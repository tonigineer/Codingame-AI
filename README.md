# Codingame-AI

A general-purpose-repo for solving coding puzzles and building competitive bots
in `Rust`.

## Workspace Commands

From the repo root:
```sh
cargo build --workspace
cargo test --workspace
cargo clippy --workspace --all-targets -- -D warnings
```
